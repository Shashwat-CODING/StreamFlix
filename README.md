Drishya is a premium Flutter-based streaming application designed to provide a seamless and high-quality entertainment experience. With a sleek user interface and powerful features, it brings together your favorite content under one roof.

## ✨ Features

- 🎥 **Movies & TV Shows**: Explore a vast library of films and series with detailed information.
- 📺 **Live TV (IPTV)**: Stream live television channels directly within the app.
- ⚽ **Sports**: Stay updated and watch live sports events.
- 🎨 **Modern UI**: Beautifully crafted glassmorphism and Material 3 design elements.
- 🎬 **Premium Playback**: High-performance player with hardware acceleration and buffering optimizations.
- 🔍 **Smart Search**: Find exactly what you're looking for with an intuitive search system.
- 🕒 **Watch History**: Keep track of what you've watched and resume seamlessly.
- 🌈 **Dynamic Colors**: Support for Android 12+ dynamic color theming.

## 🚀 Tech Stack

- **Framework**: [Flutter](https://flutter.dev)
- **State Management**: Stateful Widgets & Shared Preferences
- **Video Playback**: [media_kit](https://pub.dev/packages/media_kit)
- **Theming**: Material 3, [dynamic_color](https://pub.dev/packages/dynamic_color)
- **Animations**: [flutter_animate](https://pub.dev/packages/flutter_animate)
- **Images**: [cached_network_image](https://pub.dev/packages/cached_network_image)

## 📥 Getting Started

To get a local copy up and running, follow these simple steps.

### Prerequisites

- Flutter SDK Installed
- Android Studio / VS Code with Flutter extension
- An emulator or physical device

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/Shashwat-CODING/Drishya.git
   ```
2. Navigate to the project directory:
   ```bash
   cd Drishya
   ```
3. Install dependencies:
   ```bash
   flutter pub get
   ```
4. Run the app:
   ```bash
   flutter run
   ```
